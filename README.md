## 🛍️ Descripción

Sistema de **tienda online (e-commerce)** desarrollado con **Spring Boot y Java**, orientado a la gestión de productos, usuarios y pedidos mediante una arquitectura REST.

---

## 🚀 Funcionalidades

- 🛒 Gestión de productos (CRUD)  
- 👤 Registro y autenticación de usuarios  
- 📦 Gestión de pedidos  
- 💳 Procesamiento de pagos  
- 🔎 Búsqueda de productos  
- 📊 Panel administrativo  

---

## 🧩 Módulos

- 🛍️ Productos  
- 👤 Usuarios  
- 📦 Pedidos  
- 💳 Pagos  
- 📊 Administración
